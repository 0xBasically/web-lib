The DescriptionList component is a mimic of the `<dl>` HTML element. The element encloses a list of groups of terms (provided by the title element) and descriptions (provided by the details elements). Common uses for this element are to implement a glossary or to display metadata (a list of key-value pairs).  
Note: The Card element is used on this representation to give a structure to the list, but the raw component has no background.

## Usage

```tsx
import React from 'react';
import {Card, DescriptionList} from '@majorfi/web-lib/components';

function App(): React.ReactElement {
	return (
		<Card>
			<DescriptionList
				options={[
					{title: 'API Version', details: '0.4.2'}, 
					{title: 'Emergency shut down', details: 'NO'}, 
					{title: 'Since Last Report', details: '3 days'}, 
					{title: 'Management fee', details: '10%'}, 
					{title: 'Performance fee', details: '80%'}
				]} />
		</Card>
	);
}
```

## Properties

```tsx
type TDescriptionListOption = {
	title: string;
	details: string | ReactElement;
};
  
type TDescriptionList = {
	options: TDescriptionListOption[];
} & React.ComponentPropsWithoutRef<'dl'>;
```

### `options`
Array of pairs to display. This will be the displayed elements of this list, with the title on the left part and the details as value in bold. Elements are separated by `16px` by default.
