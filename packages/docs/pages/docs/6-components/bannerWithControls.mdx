# Banner With Controls
```ts
import {Banner} from '@yearn-finance/web-lib/components';
```

The Banner.WithControls component allows you to scroll many banners in the same space, and to close the component.

## Usage
```tsx
import React from 'react';
import {Banner} from '@yearn-finance/web-lib/components';

function	App(): React.ReactElement {
	return (
		<Banner.WithControls>
			<Banner title={'Welcome to banner'}>
				<div>{'Have a nice day.'}</div>
			</Banner>
			<Banner title={'A nice day'}>
				<div>{'You too.'}</div>
			</Banner>
		</Banner.WithControls>
	);
}
```

## Properties
```tsx
type TBanner = {
	children: ReactElement[],
	canClose?: boolean,
	onClose?: () => void
}
```

### `canClose` (optional)
Indicate if the banner can be closed. If `true`, a little close cross button will be displayed on the top right of the banner.

<CodeExample>
	<div className={'w-3/4'}>
		<Banner.WithControls canClose={false}>
			<Banner title={'Welcome to banner'}>
				<div>{'Have a nice day.'}</div>
			</Banner>
			<Banner title={'A nice day'}>
				<div>{'You too.'}</div>
			</Banner>
		</Banner.WithControls>
	</div>
</CodeExample>

```tsx {6}
import React from 'react';
```

### `onClose` (optional)
Action to perform onClose. By default, this is handled by the component itself, but you can override this action to perform a custom one.

<CodeExample>
	<div className={'w-3/4'}>
		<Banner.WithControls onClose={() => alert('hello')}>
			<Banner title={'Welcome to banner'}>
				<div>{'Have a nice day.'}</div>
			</Banner>
			<Banner title={'A nice day'}>
				<div>{'You too.'}</div>
			</Banner>
		</Banner.WithControls>
	</div>
</CodeExample>
