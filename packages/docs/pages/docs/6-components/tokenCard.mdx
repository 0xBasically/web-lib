The TokenCard is a component used to display a token image, label and value.  
A subcomponent, the TokenCard.Wrapper, is also available to multiple TokenCard together. Size of each card and grid should be controlled with the className props. The wrapper is just a div with `grid grid-cols-12 gap-4` as className.
This component extends the Card component.

## Usage
```tsx
import React from 'react';
import {TokenCard} from '@yearn-finance/web-lib/components';

function App(): React.ReactElement {
	return (
		<TokenCard.Wrapper>
			<TokenCard
				label={'Curve Rocket Pool'}
				value={'NEW ✨'}
				imgSrc={'/token.png'}
				onClick={(): void => alert('Hello!')} />
			<TokenCard
				label={'Curve CVX-ETH'}
				value={'34.02%'}
				imgSrc={'/token2.png'} />
		</TokenCard.Wrapper>
	);
}
```

## Properties
```tsx
type TTokenCard = {
	label: string;
	value: string;
	imgSrc: string;
	onClick?: React.MouseEventHandler;
} & React.ComponentPropsWithoutRef<'div'>;
```

### `onClick` (optional)
Function to call on mouse click. Enables hover and shows the chevron icon if this is set. Default to undefined. Having this props enables the hover effect.

### `className` (optional)
Custom className to provide to alter the style of the Card. This could be useful to define the size of the card in the grid. Default is `col-span-12 md:col-span-4` with a `grid-cols-12` wrapper.
