# useAccount
Remapping of the useWeb3 context, provides a more friendly limited interface for dev experience on managing accounts.

```ts
import {useAccount} from '@yearn-finance/web-lib/hooks';
```

## Usage

```tsx
import {useAccount} from '@yearn-finance/web-lib/hooks';

function	App() {
	const {address, ens, isConnected, isConnecting, isDisconnected} = useAccount();

	if (isConnecting)
		return (<div>Connecting...</div>);
	if (isDisconnected)
		return (<div>Disconnected</div>);
	if (!isConnected) 
		return (<div>Not connected</div>);

	return (
		<div>
			{ens ? ens : address}
		</div>
	);
}
```

## Return Value

```tsx
{
	address: string | null | undefined
	ens: string | undefined
	provider: ethers.providers.Provider
	isConnecting: boolean
	isReconnecting: boolean
	isConnected: boolean
	isDisconnected: boolean
	status: 'reconnecting' | 'connecting' | 'connected' | 'disconnected'
}
```

## Configuration
This hook does not have any configuration. It is just a remapping of the `useWeb3` context.  
Please note that the `useWeb3` context should be correctly configured before using this hook (either by using the `WithYearn` context, or by using the `useWeb3` one directly).
