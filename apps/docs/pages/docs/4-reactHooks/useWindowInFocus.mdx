# useWindowInFocus
This Hook can be used to detect if you dApp is focused, aka if the tab/window is focused.
Reactively track window focus with window.onfocus and window.onblur events.

```tsx
import {useWindowInFocus} from '@yearn-finance/web-lib/hooks';
```

## Usage

```tsx
import {useWindowInFocus} from '@yearn-finance/web-lib/hooks';

function	App() {
	const	hasWindowInFocus = useWindowInFocus();
}
```

## Return Value
```tsx
boolean
```
