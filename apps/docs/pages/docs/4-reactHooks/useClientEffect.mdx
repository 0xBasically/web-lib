# useClientEffect
Can be used with NextJs or any SSR system to trigger useLayoutEffect on client and useEffect on server side.

```ts
import {useClientEffect} from '@yearn-finance/web-lib/hooks';
```

## Usage

```tsx
import {useClientEffect} from '@yearn-finance/web-lib/hooks';

function	App() {
	useClientEffect((): void => {
		console.log('useClientEffect');
	}, []);
}
```

## Return Value
*NA*

## Configuration
The `useClientEffect` takes two parameters:
- the callback function to trigger
- an array of dependencies to trigger the callback function

As it's a simple remapping between [useEffect](https://reactjs.org/docs/hooks-reference.html#useeffect) and [useLayoutEffect](https://reactjs.org/docs/hooks-reference.html#uselayouteffect), the same default configuration is used.
