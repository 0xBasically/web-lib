# ABI

Return contract ABIs to use functions through ethers

- ERC20 ABI object: [link](https://github.com/yearn/web-lib/blob/main/packages/web-lib/utils/abi/erc20.abi.tsx)
- LENS ABI object: [link](https://github.com/yearn/web-lib/blob/main/packages/web-lib/utils/abi/lens.abi.tsx)

```js
import { ABI } from '@yearn-finance/web-lib/utils'

console.log(ABI.ERC20_ABI) // ABI object
console.log(ABI.LENS_ABI) // ABI object
```
