# SearchBox
```ts
import {SearchBox} from '@yearn-finance/web-lib/components';
```
<SearchBoxExample onSearch={(s) => alert(`Searching for ${s}`)} />

The SearchBox is an `<input>` wrapped in a Card with no padding. It should be used to search an element.  
The component has headless states, meaning you have to send him the useState and you can controle the callback.  

## Usage

```tsx
import {SearchBox} from  '@yearn-finance/web-lib/components';

function	App() {
	const	[searchTerm, set_searchTerm] = React.useState('');

	return (
		<SearchBox
			searchTerm={searchTerm}
			onSearch={(s: string): void => alert(`Searching for ${s}`)}
			onChange={set_searchTerm} />
	);
}
```

## Properties

```tsx
type TSearchBox = {
	searchTerm: string;
	onChange: React.Dispatch<React.SetStateAction<string>>;
	onSearch?: React.Dispatch<React.SetStateAction<string>>;
	isNarrow?: boolean;
	ariaLabel?: string;
	placeholder?: string;
};

```

### `searchTerm`
The current value typed in the SearchBox. When the user type, this value should be updated with the onChange prop.
```tsx {8}
import {SearchBox} from  '@yearn-finance/web-lib/components';

function	App() {
	const	[searchTerm, set_searchTerm] = React.useState('');

	return (
		<SearchBox
			searchTerm={searchTerm}
			onSearch={(s: string): void => alert(`Searching for ${s}`)}
			onChange={set_searchTerm} />
	);
}
```

### `onChange`
Callback when the value of the SearchBox is updated. The string send as param is directly the typed value.
```tsx {10}

function	App() {
	const	[searchTerm, set_searchTerm] = React.useState('');

		<SearchBox
			searchTerm={searchTerm}
			onSearch={(s: string): void => alert(`Searching for ${s}`)}
			onChange={set_searchTerm} />
	);
}
```

### `onSearch` (optional)
Callback when the form is submitted, trigger to perform the search.

### `isNarrow` (optional)

### `placeholder` (optional)
The placeholder attribute specifies a short hint that describes the expected value of an input field (e.g. a sample value or a short description of the expected format). The short hint is displayed in the input field before the user enters a value.

### `ariaLabel` (optional)
The ariaLabel attribute defines a string value that labels the interactive element. It's best to have because the search has no accessible name, does not accurately describe its contents, and there is no content visible in the DOM that can be associated with the object to give it meaning. Only used for accessibility. Should be a name to indicate the search. Default is set to "Search"

```tsx {8}
import {SearchBox} from  '@yearn-finance/web-lib/components';

function	App() {
	const	[searchTerm, set_searchTerm] = React.useState('');

	return (
		<SearchBox
			ariaLabel={'My Custom SearchBox'}
			searchTerm={searchTerm}
			onChange={set_searchTerm} />
	);
}
```
