The TxHashWithActions is generic component that could be used to display a Tx Hash with some defaults action. The Hash is mono-formated and can be truncated with an elipsis in the middle.
Two actions are available, copy to clipboard and open the tx in explorer.

## Usage

```tsx
import	React						from	'react';
import	{Card, TxHashWithActions}	from	'@yearn-finance/web-lib/components';

function	App(): React.ReactElement {
	return (
		<Card>
			<TxHashWithActions
				txHash={'0x56aa0161072a6dd2661adf6fbebd0cbc6f538f9ad2306cc4cbe5afa06268b2f2'}
				explorer={'https://etherscan.io'}
				truncate={8}
				className={'font-mono text-sm text-neutral-500'} />
		</Card>
	);
}
```

## Properties

```tsx
type TAddressWithActions = {
	txHash: string;
	explorer?: string;
	truncate?: number;
	wrapperClassName?: string;
	className?: string;
};
```

### `txHash`
The hash to use. It should be valid to work properly.
```tsx {8}
import	React						from	'react';
import	{Card, TxHashWithActions}	from	'@yearn-finance/web-lib/components';

function	App(): React.ReactElement {
	return (
		<Card>
			<TxHashWithActions
				txHash={'0x56aa0161072a6dd2661adf6fbebd0cbc6f538f9ad2306cc4cbe5afa06268b2f2'} />
		</Card>
	);
}
```

### `explorer` (optional)
Root uri of the explorer linked to this hash. Aka what should be the start of the URL when clicking the link-out icon. If it's not provided, the component will detect one through the Web Lib, via the `useWeb3` context, based on the current `ChainID`.
```tsx {9}
import	React						from	'react';
import	{Card, TxHashWithActions}	from	'@yearn-finance/web-lib/components';

function	App(): React.ReactElement {
	return (
		<Card>
			<TxHashWithActions
				txHash={'0x56aa0161072a6dd2661adf6fbebd0cbc6f538f9ad2306cc4cbe5afa06268b2f2'}
				explorer={'https://etherscan.io'} />
		</Card>
	);
}
```

### `truncate` (optional)
How many character should be displayed on the start and end part of the hash when truncating it. Use 0 to disable truncate action. Default is set to 5.
