The AddressWithActions is generic component that could be used to display an address with some defaults action. The address is mono-formated and can be truncated with an elipsis in the middle.  
Two actions are available, copy to clipboard and open address in explorer.

## Usage

```tsx
import	React						from	'react';
import	{Card, AddressWithActions}	from	'@yearn-finance/web-lib/components';

function	App(): React.ReactElement {
	return (
		<Card>
			<AddressWithActions
				address={'0x16388463d60FFE0661Cf7F1f31a7D658aC790ff7'}
				explorer={'https://etherscan.io'}
				truncate={5}
				className={'font-mono text-sm text-neutral-500'} />
		</Card>
	);
}
```

## Properties

```tsx
type TAddressWithActions = {
	address: TAddress;
	explorer?: string;
	truncate?: number;
	wrapperClassName?: string;
	className?: string;
};
```

### `address`
The address to use. The address should be valid and will be checksummed.
```tsx {8}
import	React						from	'react';
import	{Card, AddressWithActions}	from	'@yearn-finance/web-lib/components';

function	App(): React.ReactElement {
	return (
		<Card>
			<AddressWithActions
				address={'0x16388463d60FFE0661Cf7F1f31a7D658aC790ff7'} />
		</Card>
	);
}
```

### `explorer` (optional)
Root uri of the explorer linked to this address. Aka what should be the start of the URL when clicking the link-out icon. If it's not provided, the component will detect one through the Web Lib, via the `useWeb3` context, based on the current `ChainID`.
```tsx {9}
import	React						from	'react';
import	{Card, AddressWithActions}	from	'@yearn-finance/web-lib/components';

function	App(): React.ReactElement {
	return (
		<Card>
			<AddressWithActions
				address={'0x16388463d60FFE0661Cf7F1f31a7D658aC790ff7'}
				explorer={'https://etherscan.io'} />
		</Card>
	);
}
```

### `truncate` (optional)
How many character should be displayed on the start and end part of the address when truncating it. Use 0 to disable truncate action. Default is set to 5.
