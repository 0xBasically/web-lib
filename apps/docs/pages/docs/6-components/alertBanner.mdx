# AlertBanner
```ts
import {AlertBanner} from '@yearn-finance/web-lib/components';
```
<AlertBannerExample />

The Alert Banner component is used to display some notice. A few levels are available, info, warning, error and critical and multiple pages could be used.

## Usage
```tsx
import	React			from	'react';
import	{AlertBanner}	from	'@yearn-finance/web-lib/components';

function	App(): React.ReactElement {
	return (
		<AlertBanner
			title={'Spend your time wisely'}
			maxHeight={'max-h-[600px] md:max-h-[300px] alertBanner--wrapper'}>
			<div>
				<p>{'Yearn Vaults are a way to use technology to help manage your holdings.'}</p>
			</div>
		</AlertBanner>
	);
}
```

## Properties
```tsx
type	TAlertBanner = {
	title: string
	level?: 'none' | 'info' | 'warning' | 'error' | 'critical'
	children: ReactElement | ReactElement[]
	isVisible?: boolean
	canClose?: boolean
	onClose?: () => void
	maxHeight?: string
}
```

### `title`
Title to use for this banner. The title is a `<h4>` HTML tag.

```tsx {9-14}
}
```

### `maxHeight` (optional)
Special variable to use as className. In pure CSS, it is impossible to correctly animate height, we need to animate max-height. This allow us to customize the max-height and have a correct transition on banner close. Default is set to `300px`. Dev should adapt it to the expected height.
```tsx {9}
import	React			from	'react';
import	{AlertBanner}	from	'@yearn-finance/web-lib/components';

function	App(): React.ReactElement {
	return (
		<AlertBanner
			title={'title'}
			maxHeight={'max-h-[654px]'}>
			<div>
				{'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec euismod, nisl eget consectetur sagittis, nisl nunc egestas nisi, euismod aliquet nisi nisl euismod. Donec euismod.'}
			</div>
		</AlertBanner>
	);
}
```

### `canClose` (optional)
Can the banner be closed? Default is set to true. If false, the banner will alwasy render and the cross will be hidden.

### `onClose` (optional)
Action to perform onClose. By default, this is handled by the component itself, but you can override this action to perform a custom one.

<CodeExample>

		title={'Spend your time wisely'}
		onClose={() => alert('nop')}>
